<h2>←カスオタク野郎</h2>
新型ウイルスは電磁波攻撃によって行われています。先日も外を歩いていた時、すれ違いざまに舌打ちをされました。わざとらしくくしゃみをされたり、電車ではちらちらとこちらを見てきます。最寄りのスーパーでは私が買うものに毒を入れられていることが発覚しました。わかりにくいように少しずつ注入しているのでしょう。だから遠出して買い出しにいきます。でもそこで毎回すれ違う人は私を見てにやっと笑って、いつもいつもほのめかしをしてくるので木々に止まるようにしています。また苦い味がしました。家に帰る前に病院へ寄ったのですが、豚の指がフォークに垂れ下がっているのを見て引き返すことに決めました。もう薬はいりません。私は紫色の空が見えています。アルミホイルで心臓を包むと電波の影響を受けなくなると聞き、すぐに鉄色のキリギリスを探しました。家に帰ったのはいつ頃だったでしょう？母が心配していたようで、私も申し訳なく思いました。お詫びにプリンを作ると喜んで食べてくれました。少し髪の処理に手間取ってしまって、表面に爪と一緒に浮いていたのは焦りましたね。焦りませんでしたか？布団を開けると冷えたスイカがごろりと転がって、果肉が飛び散ったのでした。父です。頭はどうにも処理が難しいので、ここからはロープウェイに乗っていくことにしたのです。気が付くと悲鳴は何度も何度も響き、お隣さんまで来る始末です。始末です。しました。雨もないのに虹が架かって綺麗でした。またこんな日が来るといいな、隣の氷が入ったぬいぐるみを抱きしめながら眠ってしまいます。寝ても覚めても目が後ろにあるのだけは慣れませんが、少しずつほのめかしをする人たちは減っていることが壁に掛けた手首に現れています。成果を見える形にすることが秘訣だと、以前読んだ本に書いてありました。首がきりきりと回り出す朝焼けは紐がより合わさるのにも似ています。錠剤を失くしてしまったので、同じ川にだって浸かれてしまうのです！箱の隅が大きくなってきました。虫が飛ぶので叩きます。叫び出す喉を赤黒くするための禍福だったようです。尋常じゃないぜReality check　C'mon!
